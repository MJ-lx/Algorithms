## 目录

```js

 |-- src
    |-- DynamicConnectivity // 动态连接
      |-- actualCombat.ts // 算法的解决实际问题示例
      |-- pathQuickUnion.ts
      |-- quickFind.ts
      |-- quickUnion.ts
      |-- weightQuickUnion.ts
      |-- __test__ // 测试文件

```

## 使用说明
- npm i 安装相应依赖
- npm run test <path> 执行对应的测试文件
